## Purpose
- Each SCHOLAR token represents ₹1 of scholarship
- Max 10,000 tokens (₹10,000) per student per month
- Max 50,000 tokens (₹50,000) monthly treasury pool

## Student Flow
1. Student opts-in to SCHOLAR ASA
2. AI Agent evaluates eligibility
3. Smart contract authorizes transfer
4. Treasury wallet sends SCHOLAR tokens atomically
5. Transaction logged immutably on-chain
